# Структура проєкту

## Основні HTML сторінки

- **index.html** - Головна сторінка "Про мене"
  - Відображає інформацію про дизайнера
  - Можливість редагування через CMS
  - Кнопка автентифікації

- **portfolio.html** - Сторінка портфоліо
  - Галерея робіт
  - Кнопка "Контакт" з модальним вікном

- **services.html** - Сторінка послуг
  - Опис послуг
  - Кнопка переходу на сторінку замовлення

- **order.html** - Сторінка оформлення замовлення
  - Форма з даними клієнта
  - Вибір тарифу
  - Підтвердження замовлення

## JavaScript файли

- **js/main.js** - Завантаження контенту для головної сторінки
- **js/portfolio.js** - Функціональність портфоліо та модального вікна
- **js/services.js** - Завантаження контенту послуг
- **js/order.js** - Логіка оформлення замовлення та вибору тарифів

## CSS

- **css/style.css** - Всі стилі для проєкту
  - Адаптивний дизайн
  - Модальні вікна
  - Анімації та переходи

## Netlify CMS

- **admin/index.html** - Адмін панель CMS
- **static/admin/config.yml** - Конфігурація CMS з усіма колекціями

## Контент (content/)

- **about/index.json** - Інформація про дизайнера
- **services/index.json** - Опис послуг
- **contacts/index.json** - Контактна інформація
- **portfolio/** - Роботи (створюються через CMS)
- **tariffs/** - Тарифи (створюються через CMS)
- **orders/** - Замовлення (створюються через форму)

## Статичні файли

- **static/img/** - Зображення, завантажені через CMS

## Налаштування

- **netlify.toml** - Конфігурація Netlify
- **.gitignore** - Файли для ігнорування Git

## Документація

- **README.md** - Повна документація
- **QUICKSTART.md** - Швидкий старт
- **STRUCTURE.md** - Цей файл
