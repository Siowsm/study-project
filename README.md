# Портфоліо дизайнера з Netlify CMS

Повнофункціональне портфоліо дизайнера з системою управління контентом через Netlify CMS.

## Особливості

- ✅ Головна сторінка "Про мене" з можливістю редагування тексту, фото та GIF
- ✅ Сторінка портфоліо робіт з модальним вікном контактів
- ✅ Сторінка послуг з описом та кнопкою оформлення замовлення
- ✅ Сторінка оформлення замовлення з формою та вибором тарифів
- ✅ Система автентифікації через Netlify Identity
- ✅ Адмін панель для редагування всього контенту
- ✅ Можливість додавання тарифів (тільки для адміністратора)
- ✅ Сучасний адаптивний дизайн

## Структура проєкту

```
.
├── index.html              # Головна сторінка "Про мене"
├── portfolio.html          # Сторінка портфоліо робіт
├── services.html           # Сторінка послуг
├── order.html              # Сторінка оформлення замовлення
├── css/
│   └── style.css          # Стилі для всіх сторінок
├── js/
│   ├── main.js            # JavaScript для головної сторінки
│   ├── portfolio.js       # JavaScript для портфоліо
│   ├── services.js        # JavaScript для послуг
│   └── order.js           # JavaScript для замовлення
├── admin/
│   └── index.html         # Netlify CMS адмін панель
├── static/
│   ├── admin/
│   │   └── config.yml     # Конфігурація Netlify CMS
│   └── img/               # Папка для зображень
├── content/               # Контент (створюється через CMS)
│   ├── about/
│   ├── portfolio/
│   ├── services/
│   ├── tariffs/
│   └── contacts/
├── netlify.toml           # Налаштування Netlify
└── README.md              # Документація
```

## Встановлення та запуск

1. Клонуйте репозиторій або завантажте файли
2. Відкрийте `index.html` у браузері для локального перегляду

## Деплой на Netlify

1. Зареєструйтеся на [Netlify](https://www.netlify.com/)
2. Підключіть ваш GitHub/GitLab репозиторій
3. Netlify автоматично визначить налаштування з `netlify.toml`
4. Після деплою, перейдіть на `/admin` для доступу до CMS

## Налаштування Netlify CMS

### Перший вхід

1. Після деплою на Netlify, перейдіть на `your-site.netlify.app/admin`
2. Натисніть "Enable Identity" в налаштуваннях Netlify
3. Увімкніть "Git Gateway" для доступу до репозиторію
4. Зареєструйтеся або увійдіть через Netlify Identity
5. Почніть додавати контент через CMS інтерфейс

### Редагування контенту

Через адмін панель (`/admin`) ви можете:

- **Про мене**: Редагувати заголовок, опис, фото та GIF
- **Портфоліо**: Додавати нові роботи з назвою, описом та зображенням
- **Послуги**: Редагувати заголовок та опис послуг
- **Тарифи**: Додавати та редагувати тарифи (тільки для адміністратора)
- **Контакти**: Оновлювати контактну інформацію та посилання на соціальні мережі
- **Замовлення**: Переглядати всі замовлення та змінювати їх статус

## Функціональність

### Головна сторінка (index.html)
- Відображення інформації про дизайнера
- Можливість редагування через CMS
- Кнопка "Створити акаунт" для входу/реєстрації

### Сторінка портфоліо (portfolio.html)
- Галерея робіт з ефектом наведення
- Кнопка "Контакт" що відкриває модальне вікно з контактами
- Можливість додавання робіт через CMS

### Сторінка послуг (services.html)
- Опис послуг з можливістю редагування
- Кнопка "Хочу оформити замовлення" що веде на сторінку замовлення

### Сторінка замовлення (order.html)
- Форма з полями: ім'я, прізвище, місто, вулиця, дім, телефон
- Після заповнення форми - вибір тарифу
- Можливість додавання нових тарифів через CMS (тільки адміністратор)
- Підтвердження замовлення

## Безпека та автентифікація

- Використовується Netlify Identity для автентифікації
- Тільки авторизовані користувачі можуть редагувати контент через CMS
- Паролі та логіни зберігаються безпечно через Netlify Identity
- Адміністратор може налаштувати доступ до редагування через налаштування Netlify

## Кастомізація

### Зміна кольорів
Відредагуйте CSS змінні в `css/style.css`:
```css
:root {
    --primary-color: #4a90e2;
    --secondary-color: #50c878;
    /* ... */
}
```

### Додавання нових полів
1. Додайте поля в HTML форму
2. Оновіть валідацію в JavaScript
3. Оновіть `static/admin/config.yml` для відображення в CMS
